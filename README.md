# PolishPoint

Marketing website for **PolishPoint** — all-in-one cleaning business management software you build once and own forever. A product by Saasassins.

## Stack

Static HTML / CSS / JS. No build step required.

## Local development

```bash
ruby -run -e httpd . -p 3000
# or
python3 -m http.server 3000
```

Then open http://localhost:3000

## Structure

- `index.html` — Home page
- `about.html` — About page
- `blog.html` — Blog page
- `css/style.css` — All styles
- `js/main.js` — Interactivity (nav, reveals, calculator, FAQ, Calendly)
- `images/` — Logos and device preview screenshots
